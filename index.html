Hello, I am Hannah McCormack, a user centric designer. I specialize in designer process that includes user research, ideation, prototyping and user testing and handing over the project to front end developers. I take pride in bridging the gap between developers and end users.

My Skills

USER RESEARCH

I believe finding the problem resonates with testing your users and understanding the empathy behind what is complicating things. I believe taking all data and analyzing it together to pinpoint the true problem, find a solution, and create an impact!

PROTOTYPING

Once I identify a problem and create sketches, I begin to build wire frames. Conduct usability testing and gather feedback and iterate to create my final prototype.

INFORMATION ARCHITECTURE

Creating a user flow and then testing our users on the task to justify if the navigation is as simple to use as we hope. Design or redesign a site map for both header and footer to show simplified version of a complicated website and conduct usability testing.
